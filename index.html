CSProfile

Privacy Policy • Last updated May 2026

Overview

CSProfile is a browser extension that reads public Steam profile data and allows users to send profile information to their own Discord server through a configured webhook.


We do not collect, store, sell, or share personal data with the extension developer or any third party.

Stored Data

What We Store

  • Discord webhook URL (stored locally using chrome.storage.local)
  • Extension preferences and configuration options

This information remains on your device and is never transmitted to the developer.

Accessed Data

Steam Information

When viewing public Steam profiles, CSProfile may access:

  • Display Name
  • Avatar Image
  • SteamID64
  • CS2 Statistics
  • Current Game Status

This information is processed temporarily and only for generating the Discord embed requested by the user.

Third-Party Services

Services Used

  • Cloudflare Workers — Used as a relay for Steam and FACEIT public data requests.
  • Steam API — Retrieves public Steam profile and statistics data.
  • FACEIT API — Retrieves public FACEIT rank information.
  • Discord — Receives embed messages only when the user explicitly clicks "Send to Discord".
Permissions

Why Permissions Are Required

Permission Purpose
activeTab Read the currently open Steam profile page.
storage Store user preferences locally.
scripting Inject the CSProfile interface into Steam pages.
tabs Detect whether the current tab is a Steam profile.
Children's Privacy

CSProfile is not directed toward children under the age of 13. We do not knowingly collect information from children.

Policy Updates

Any future updates to this Privacy Policy will be published on this page together with a revised update date.

Contact

Questions or concerns can be submitted through the project's GitHub repository or by contacting the developer directly.
